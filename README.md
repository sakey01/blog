## Personal Blog Page

A personal blog ranging from from everyday shower thoughts to general tech tips.

## Built with
- React
- Tailwind CSS
- Vite
- Firebase

## Features
- Sign up / Sign in with Firebase Auth
- Upload and view public posts
- Authors for posts
- A like system to see what's popular
  
## Dependencies
- tailwindcss
- heroicons
- firebase
- react-hot-toast
- react-router-dom
